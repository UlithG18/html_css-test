
# Nexcent – Landing Page

## Project 

The main objective of this project is to create a clean, accessible, and responsive interface, faithfully replicating the layout, structure, colors, typography, and behavior of the original design using only HTML and CSS

It follows the design provided in Figma and applies modern layout techniques such as **Flexbox** and **responsive design principles**



## How to Clone and View the Project

1. Clone the repository from GitHub:

   ```bash
   git clone https://github.com/UlithG18/html_css-test
   ```

2. Open the project folder

3. Open the `index.html` file in your web browser
